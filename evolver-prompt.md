# Evolver 主页优化提示词

## 目标
将个人主页优化到与 https://evomap.ai/ 相同水平

## 参考资源
- Prompt Engineering Guide: https://github.com/dair-ai/Prompt-Engineering-Guide (71k stars)
- Context Engineering: https://github.com/Meirtz/Awesome-Context-Engineering

## 优化方向

### 1. UI/UX 设计
- 参考 EvoMap 的深色科技风格
- 添加更多微交互和动画
- 优化排版和间距
- 添加渐变和光效

### 2. 内容丰富
- 添加更多项目案例
- 添加技术博客/文章链接
- 添加个人成就/奖项
- 添加技能详细描述

### 3. 技术实现
- 添加滚动动画 (Intersection Observer)
- 添加鼠标跟随效果
- 优化加载性能
- 添加暗色/亮色模式切换

### 4. 参考风格
- Rajdhani / Space Grotesk 字体
- 蓝紫渐变配色
- 毛玻璃效果卡片
- 浮动光球背景

## 执行要求
1. 每次修改需要提交 git
2. 保持代码简洁可维护
3. 确保移动端适配
4. 推送更新到 GitHub
